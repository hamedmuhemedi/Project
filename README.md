## Description:
A responsive web application for real-time currency conversion using live exchange rates. This tool allows users to convert between multiple currencies instantly, save conversion history, and toggle between light/dark modes.

## Features:
1. **Real-Time Conversion** between 14+ popular currencies
2. **Live Exchange Rates** from ExchangeRate-API
3. **Conversion History** stored in localStorage
4. **Dark/Light Mode** toggle
5. **Auto-Convert** on input change
6. **Swap Currencies** with one click
7. **Responsive Design** for all devices
8. **Rate Refresh** button for latest rates

## Technologies Used:
- **Frontend:** HTML5, CSS3, JavaScript
- **UI Framework:** Bootstrap 5
- **Icons:** Font Awesome 6
- **API:** ExchangeRate-API (free tier)
- **Storage:** Browser localStorage

## How to Run:
1. Save all files in a folder
2. Open `index.html` in any modern browser
3. (Optional) Get a free API key from [ExchangeRate-API](https://www.exchangerate-api.com/) and replace `YOUR_API_KEY_HERE` in `script.js`
